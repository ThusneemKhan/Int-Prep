Diff. between Div and Span?

Div: - it is a blocklevel element, it is bound to take the whole width of the screen.
- it is used for grouping of elements in HTML documents, means div should be used to wrap section of a document.

Span: - it is a inline element, means it only takes the place that is required for it.
- it is used to wrap small portion of text and images etc.

How many types of lists?

There are three different types of list in HTML

Ordered list — Used to create a list of related items, in a specific order.
Description list — Used to create a list of terms and their descriptions.
